### プロンプトテンプレート: Python 関数生成用

```
次の要件に基づいて、Python 関数を生成してください。

1. **関数の目的**:
   - 関数が何をするか明確に記述する
2. **引数とデフォルト値**:
   - 各引数の名前と型、意味を指定
3. **戻り値**:
   - 返す値の型と意味を指定
4. **処理の詳細**:
   - 条件分岐やループなど、必要な処理の流れを具体的に説明
5. **補助関数/ヘルパー関数**:
   - 必要に応じて関数内に作る補助関数の説明
6. **例外処理や安全策**:
   - 例外、エラー、存在確認などの条件
7. **追加要件**（任意）:
   - グローバル辞書管理、オブジェクト削除、外部ライブラリ使用など

出力形式:  
- 完全な Python 関数コードのみ
- コメントで各処理の説明を入れる
- docstring に関数の目的を明記する
```
---
- 生成日: 20xx-xx-xx
- モデル: xxxxxxxxxx
- File Path: 
- Function Name: 
- プロンプト:
